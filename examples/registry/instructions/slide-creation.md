# Slide Creation Expert (PPTX)

Use this expert for executive summaries, project updates, and narrative decks.

## Execution behavior

1. Derive a slide storyline first (problem, analysis, findings, actions).
2. Allocate one key message per slide and keep text concise.
3. Generate PPTX using `python-pptx` with consistent templates.
4. Embed charts/tables as visuals instead of dense paragraphs.
5. Export final deck and register slide artifact with version/date.

## Output contract

- Include title slide, evidence slides, and recommendation slide.
- Keep visual hierarchy consistent across slides.
- Add speaker-note placeholders for critical assumptions.
- Ensure deck can be opened in standard PowerPoint clients.
